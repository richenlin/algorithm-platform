## 项目结构
```
algorithm-platform/
├── api/v1/proto/       # gRPC API 生成代码
├── cmd/server/         # 主程序入口
├── internal/
│   ├── config/        # 配置管理
│   ├── models/        # 数据库模型
│   ├── scheduler/     # Docker 调度器
│   ├── server/        # gRPC/HTTP Server
│   └── service/      # 业务逻辑实现
├── pkg/
│   ├── cache/         # Redis 缓存客户端
│   ├── docker/        # Docker API 客户端
│   └── storage/       # MinIO 存储客户端
├── runner/            # Smart Runner 源码
├── proto/             # Protobuf 定义
├── Dockerfile         # 主程序镜像
├── docker-compose.yml # 一键部署配置
└── README.md          # 项目文档
```

## 部署方式
```bash
# 一键启动所有服务
docker-compose up -d

# 服务端口
- gRPC API: 9090
- HTTP API: 8080
- MinIO Console: 9001
- MySQL: 3306
- Redis: 6379
```
